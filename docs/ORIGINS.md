# Origins

Haunted Studio was first prototyped inside
[`Fevriergirl/multi-agent-a2a-orchestration`](https://github.com/Fevriergirl/multi-agent-a2a-orchestration).
That repository began as a different proposed A2A orchestration project and was
not a clear permanent boundary for Haunted Studio.

The audited migration source was branch `cleanup/haunted-studio-identity` at
commit `926802dda947155212169f3e362e9a29c1b1b721`, reviewed in
[pull request #1](https://github.com/Fevriergirl/multi-agent-a2a-orchestration/pull/1).

This repository is the clean standalone continuation:

- only files belonging to Haunted Studio were copied;
- the former repository's `.git` directory and generated data were excluded;
- source-repository audit files and the archived A2A proposal were not copied;
- a fresh Git history begins at standalone version `0.1.0`; and
- the original repository remains unchanged as historical provenance.

Prototype milestones labeled `0.2.0` and `0.3.0` were internal build labels,
not prior releases of this standalone repository.

This lineage does not support claims of consciousness, subjective experience,
personhood, or demonstrated artistic development. It documents software
provenance only.
